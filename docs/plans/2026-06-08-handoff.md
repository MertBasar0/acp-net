## Onemli Bulgular

1. Gemini CLI ACP `--acp` bayragi ile calisiyor. `--experimental-acp` deprecated.
2. Gemini gercek dogfood basarili oldu.
3. WSL agent icin ACP payload path'leri de normalize edilmeli; bu nedenle `ToAgentPath` eklendi.
4. Gemini ortaminda `rg` yok:
   - `preflight.tool.missing rg`
   - Gemini stderr: `Ripgrep is not available. Falling back to GrepTool.`
5. `git` ve `node` WSL runtime icinde bulundu.

## Yarin Icin Onerilen Siradaki Is

Sıradaki en mantikli is:

**Spike 008: Environment Policy + Fail Fast**

Sorular:

1. `RequiredExecutables` eksikse runner sadece transcript uyarisi mi yazmali?
2. Yoksa opsiyonel `PreflightFailurePolicy.Warn | Throw` gibi policy mi olmali?
3. `rg` eksikligi icin urun ne onermeli?
   - system install: `sudo apt install ripgrep`
   - `AdditionalPathEntries`
   - tool bundle discovery
4. Gemini dogfood sample'i bu policy ile nasil davranmali?

Onerilen API:

```csharp
PreflightFailurePolicy = AcpPreflightFailurePolicy.Warn
```

veya:

```csharp
RequiredTools = [
    AcpRequiredExecutable.Warn("rg"),
    AcpRequiredExecutable.Throw("git")
]
```
